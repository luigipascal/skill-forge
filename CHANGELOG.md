# Changelog

All notable changes to skill-forge.

## [0.1.0] — 2026-06-30

### Added

- `validate` — lint `SKILL.md` against agentskills.io-style rules
- `init` — scaffold project or personal skills
- `convert` — migrate Cursor intelligent rules and commands to skills
- `install-curated` — install Apache 2.0 skills from anthropics/skills plus bundled examples
- `doctor` — environment and installed-skill health check
- Example skills: `typescript-quality`, `mcp-server-quality`, `orchestra-memory`
- GitHub Actions workflow to validate skills on push/PR
